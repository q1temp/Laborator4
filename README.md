## Задача 1

### Текст задачи

Дерево содержит вещественные числа. Отбросить в каждом из них дробную часть

### Алгоритм решения

1) Попадаем в функцию main
2) Задаем количество узлов и вызываем функцию generateRandomTree nodeCount для построения исходного дерева
3) В функции, если переданное кол-во узлов count <= 0, то возвращаем Empty - пустое дерево. Иначе генерируем рандомное вещественное число по формуле rnd.NextDouble() * 20.0 - 10.0 (диапазон от –10.0 до 10.0). Рекурсивно вызываем generateRandomTree (count - 1) для получения дерева с оставшимися узлами. Вставляем value в полученное дерево с помощью insert value. Возвращаем получившееся дерево. 
4) Функция insert. Принимает значение value и дерево tree. Если дерево пустое (Empty) – создаёт новый узел Node(value, Empty, Empty). Иначе, если value < v (где v – значение текущего узла), рекурсивно вставляет value в левое поддерево и возвращает Node(v, newLeft, right). В противном случае (если value >= v) вставляет в правое поддерево и возвращает Node(v, left, newRight). В результате получаем упорядоченное бинарное дерево. 
5) Получаем root исходное дерево с вещественными числами.
6)Вызываем printTree "" root. Если встретили Empty, ничего не печатаем (возврат).
Если встретили Node(v, left, right), то печатаем текущее значение с отступом, затем рекурсивно печатаем левое поддерево с увеличенным отступом (indent + " "), затем правое поддерево.
7) Вызываем  mapTree int root. Если дерево пустое, то возвращаем Empty. Если узел применяем f к значению узла и рекурсивно вызываем mapTree для левого и правого поддеревьев, собирая новый узел Node(f(data), mapTree f left, mapTree f right). 
8) Выводим новое дерево с целыми числами.
9) Завершаем программу
